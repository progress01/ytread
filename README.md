# ytread
單一 HTML 檔案的影片筆記工具，整合了影片播放器與文字編輯區。設計目的在於減少手部離開鍵盤的頻率，透過快捷鍵控制影片播放與插入時間戳記標籤。

## 功能概述

* **影片支援**：支援 YouTube（透過 API 控制）與 Bilibili（透過 iframe 嵌入）。
* **鍵盤控制**：提供播放、暫停、快進、快退及變速的鍵盤快捷鍵。
* **時間戳記**：在筆記中插入標籤時會自動附帶當前影片時間碼。
* **雙擊跳轉**：雙擊筆記區內的時間碼（例如 01:23），影片會跳轉至該時間點（僅限 YouTube）。
* **自動章節**：嘗試自動抓取 YouTube 影片說明欄中的時間軸章節並匯入。
* **匯出功能**：將筆記內容整理為 Markdown 格式並複製到剪貼簿，包含統計資訊。
* **背景音樂**：提供 BGM 模式，可循環播放指定的 YouTube 播放列表。

## 使用方式

1.  將 HTML 檔案下載至本機。
2.  使用現代瀏覽器（Chrome, Edge, Firefox, Safari）開啟該檔案。
3.  在頂部欄位輸入影片網址並點擊「載入」。

## 快捷鍵列表

本工具使用 `Alt` 鍵作為組合鍵，避免與瀏覽器預設功能衝突。

### 影片控制（右手區）

* **Alt + K**：播放 / 暫停
* **Alt + J**：倒退 10 秒
* **Alt + L**：快進 10 秒

### 快速標籤（左手區）

插入標籤時會自動換行並加上時間戳記。

* **Alt + 1**：[重點]
* **Alt + 2**：[疑問]
* **Alt + 3**：[待辦]
* **Alt + 4**：[測試]
* **Alt + 5**：[其他]

## 介面說明

* **頂部欄**：輸入筆記標題、影片網址及 BGM 設定。
* **左側/上方**：影片播放區域。
* **右側/下方**：筆記編輯區域、控制按鈕列、匯入/匯出工具。
* **底部**：簡易操作提示。

## 注意事項

* Bilibili 影片因平台限制，不支援外部控制（快捷鍵無法控制播放進度，亦無法雙擊跳轉），僅提供觀看與筆記功能。
* 自動抓取章節功能依賴第三方 API，若 API 失效或影片無說明欄資訊，需手動使用匯入功能貼上時間軸文字。
* 本工具為純前端運作，所有資料暫存於瀏覽器當前頁面，關閉或重新整理頁面後內容將會消失，請記得定期使用匯出功能保存筆記。

## 系統需求

* 支援 HTML5 與 JavaScript 的網頁瀏覽器。
* 需連接網際網路以載入 YouTube/Bilibili 播放器資源。
